GIT
| Windowns  | Diferença Dos Comandos |  Unix = Bash |
| :---        |    :----:   |          ---: |
| - cd = Voltar 1 nivel     |   Igual     | -cd = Voltar 1 nivel   |
| - dir = Listar  |   Igual     | - ls = Listar   |
| - mkdir = Criar Diretorio  |   Igual     | - mkdir = Criar Diretorio   |
